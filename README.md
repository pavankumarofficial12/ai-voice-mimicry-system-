# AI Voice Mimicry System

An advanced AI system that **mimics human voice** using Coqui XTTS v2. Upload a voice sample and get a highly realistic synthetic voice that speaks any text while preserving tone, pitch, and speaking style.

---

## ✨ Features

- Voice cloning using XTTS v2
- Speech-to-Text with Whisper
- Prosody analysis (pitch, energy, duration)
- Voice similarity scoring (Resemblyzer)
- High-quality multilingual voice generation
- Public audio output access

---

## 🛠 Tech Stack

- **Framework**: FastAPI
- **Voice Cloning**: Coqui TTS (XTTS v2)
- **Speech Recognition**: OpenAI Whisper
- **Speaker Embedding**: Resemblyzer
- **Audio Processing**: Librosa + PyDub
